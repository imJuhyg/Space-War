## 게임방법
방향키(W, A, S, D)로 전투기를 조작하고 <SPACE BAR>와 <K> 키로 총알과 미사일을 발사할 수 있습니다.
  
** 한글 입력 모드 상태에서는 조작되지 않을 수도 있습니다.
